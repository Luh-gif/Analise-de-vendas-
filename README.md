## 01_Analise_de_Vendas
**Objetivo:**  
Analisar os dados de vendas de uma empresa fictícia para gerar insights sobre performance de produtos, receita e tendências de vendas.  

**Etapas realizadas:**  
- Carregamento e exploração dos dados brutos (`Dados de vendas Brutos.csv`).  
- Limpeza e transformação de dados para análise (`Dados de vendas traduzidos.csv` e Excel).  
- Análise exploratória utilizando gráficos com matplotlib e seaborn.  
- Treinamento de modelo de regressão linear para previsão de vendas futuras (`modelo_vendas.pkl`).  
- Criação de relatório final em Word e Dashboard em Power BI.  

**Arquivos importantes:**  
- `analise_vendas.ipynb` – Notebook com toda a análise e modelagem.  
- `modelo_vendas.pkl` – Modelo de regressão treinado.  
- `Relatório de Análise de Vendas.docx` – Documento com insights.  
- `Dados de vendas Brutos.csv` / `Dados de vendas traduzidos.csv` – Bases de dados utilizadas.  

**Tecnologias / Bibliotecas:**  
`pandas`, `numpy`, `matplotlib`, `seaborn`, `LinearRegression`, `train_test_split`, `joblib`, `tkinter`, `askopenfilename`.
Os dashbords estao dentro do  `Relatório de Análise de Vendas.docx` – Documento com insights.
